# Tic Tac Toe

A simple Tic Tac Toe game built with React and TypeScript. This project is designed for learning and practicing modern React concepts, including hooks, component composition, and state management.

## Features

- 3x3 interactive game board
- Player name editing
- Game history log
- Win and draw detection
- Responsive design

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Start the development server:**
   ```bash
   npm run dev
   ```
3. **Open your browser:** Navigate to `http://localhost:5173` (or the port shown in your terminal)
